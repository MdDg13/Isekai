# Texture Assets

Generated assets live under `public/generated/<category>/<id>.png`. They are created via the Workers AI manifest (`assets/ai/manifest.json`) using `npm run generate:ai`. Each entry below is appended automatically after a successful generation run.

> Regenerate a specific entry with `npm run generate:ai -- --id=<asset_id>`.
